= ENDPOINT-DEACTIVATE(1)

== NAME

endpoint-deactivate - Remove authentication credentials

== SYNOPSIS

*endpoint-deactivate*
.BI [ "pattern" "...]"
.br

== DESCRIPTION

*endpoint-deactivate *
removes credentials previously assigned to a logical endpoint by 
*endpoint-activate. *
If pattern is not specified, it defaults to '*'.

== EXAMPLES

.TP
Deactivate all endpoints:
*endpoint-deactivate **
.TP
Deactivate endpoints with names matching a pattern:
*endpoint-deactivate testendpoint**
