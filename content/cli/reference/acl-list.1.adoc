= ACL-LIST(1)

== NAME

acl-list - Show all Access Rules on an endpoint

== SYNOPSIS

acl-list 'ENDPOINT' 

== DESCRIPTION

Display all rules in an endpoint's access control list (ACL).

The current output format is JSON.  Paths are URL-encoded.

include::include/more_acl.adoc[]


== EXAMPLES

Show all ACL rules for the endpoint "myep":

----terminal
acl-list bob#myep
----terminal
