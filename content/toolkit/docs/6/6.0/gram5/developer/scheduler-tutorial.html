Abstract:

GRAM provides a scheduler-independent abstraction to remote job management. The scheduler abstraction is divided into three sections: job submission and cancelling, job monitoring, and information publication. This set of tutorials describes how to implement and bundle all packages needed for a complete scheduler interface for GRAM.
